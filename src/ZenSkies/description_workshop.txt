[h1]✦ Zen's Sky ✦[/h1]
A highly configurable, and cross-compatible mod that vastly improves the visual style of the sky in various ways.
As well as including simple and easy to use menu configuration straight from the titlescreen; you don't even have to click save!

[h2]✦ Features ✦[/h2]

[b]Currently adds:[/b]
[list]
    [*]More dynamic sun in the style of 1.4.5 leaks.
    [*]High-res dynamic moon, works for every vanilla moon style (Including Calamity Fables's 16 additional moon styles).
    [*]Additional support for adding your own high-res textures to this mod with simple crossmodding.
    [*]Prettier stars, with 5 styles to choose from.
    [*]Supernovae.
    [*]Fancy shooting stars.
    [*]Wind particles.
    [*]Dynamic cloud lighting.
    [*]Menu configuration via a dropdown.
[/list]

[b]Compatibility with big mods like:[/b]
*'Macrocosm [BETA]' by MechanicPluto24
    (Currently no high-res earth assets, read below if you would like to contribute.)
'Realistic Sky' by Lucille Karma,
*'Wrath of the Gods' by Lucille Karma
    (Currently no changes to telescope/stargazing scenes, read below if you would like to contribute.),
'Lights And Shadows' by yiyang233,
'Calamity Fables' by IbanPlay.

[b]And even small mods like:[/b]
'Calamity Mod' by Ozzatron,
'Red Sun and Realistic Sky' by Waffles22,
'IDG's Better Night Sky' by IDGCaptainRussia and Trivaxy.

[h2]✦ Misc ✦[/h2]

[u]This mod is client-sided*. But will sync over data if both the server and client have it enabled.[/u]

This mod provides a number of miscellaneous Mod.Call implementations for simpler compat;
You can find classes anotated with ModCall [url=https://github.com/search?q=repo%3AZenTheMod%2FZensSky%20ModCall&type=code]​Here[/url].

You can report any issues with the mod [url=https://github.com/ZenTheMod/ZensSky/issues]​Here[/url].
